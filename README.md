# Server

Server for registering a user

### Requirements

1. Mongo DB
2. Redis

### NPM Scripts For Deployment

- Install dependencies

  ```bash
  npm install
  ```

- Spin Up Databases

  ```bash
  npm run start:dev
  ```
- Postman

